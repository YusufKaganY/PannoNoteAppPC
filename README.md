📌 Panno Widget App
Panno, masaüstünde her an elinizin altında olan, hafif (lightweight), şık ve yüksek performanslı bir not alma ve görev yönetim asistanıdır. Yazılımcılar, öğrenciler ve odaklanmış bir çalışma ortamı arayanlar için WPF (.NET Framework) ve SQLite altyapısı ile geliştirilmiştir.

🚀 Öne Çıkan Özellikler
🥷 Ninja Modu (Global Hotkey): Başka bir programda veya oyunda olsanız bile Ctrl + Shift + P kısayolu ile Panno'yu anında ekrana çağırabilir veya gizleyebilirsiniz.

🌓 Dinamik Temalar: Modunuza göre Sarı, Mavi, Yeşil, Pembe veya göz yormayan Karanlık Tema seçeneklerinden birini kullanın.

👻 Hayalet Modu: Saydamlık ayarı sayesinde Panno'yu masaüstü duvar kağıdınızla bütünleştirin.

📋 Tek Tıkla Kopyala: Notlarınızı seçmekle uğraşmayın, tek bir butonla panoya (clipboard) kopyalayın.

🖱️ Akıllı Sürükle-Bırak: Görevlerinizi farenizle sürükleyerek öncelik sırasına koyun.

↩️ Geri Al (Undo): Yanlışlıkla sildiğiniz notları saniyeler içinde hayata döndürün.

💾 Yerel Veritabanı: Tüm verileriniz bilgisayarınızda, size özel SQLite veritabanında güvenle saklanır.

📌 Ekrana Sabitleme: "Topmost" özelliği ile Panno'yu her zaman diğer pencerelerin üstünde tutun.

🛠️ Teknik Detaylar
Bu proje, modern Windows masaüstü uygulama geliştirme prensipleri kullanılarak inşa edilmiştir:

Framework: .NET Framework 4.8 (WPF)

Veritabanı: SQLite (Microsoft.Data.Sqlite)

UI/UX: XAML tabanlı dinamik kaynak (DynamicResource) yönetimi ve akıcı animasyonlar.

Windows API Integration: * user32.dll ile Global Hotkey yönetimi.

System.Windows.Forms ile System Tray (Sistem Tepsisi) entegrasyonu.

Windows Registry ile "Başlangıçta Çalıştır" desteği.

📥 Kurulum ve Kullanım
